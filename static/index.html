Análisis Exploratorio de Datos (EDA)

Contexto y Limpieza de la Base de Datos

El corpus OSINT base consistió en miles de alertas tempranas, noticias y registros de eventos geopolíticos extraídos de fuentes públicas como GDELT y repositorios ACLED. Durante la fase inicial se detectó "ruido" significativo típico de la ingesta web masiva. El 12% de los registros presentaban valores nulos en campos críticos como coordenadas geográficas o texto del evento. Asimismo, se identificaron valores atípicos (outliers): reportes con coordenadas ilógicas o longitudes de texto que indicaban errores de web scraping (textos vacíos o artículos de miles de palabras que desviaban la vectorización).

Resolución: Se aplicó una poda estadística. Los nulos espaciales fueron imputados si existía metadato regional (ej. "Teherán"), o eliminados. Los textos se estandarizaron mediante tokenización, conversión a minúsculas y eliminación de stopwords, permitiendo que la base quedara 100% íntegra para el modelado de predicción de riesgo.

Objetivo del Proyecto

OSINT Intelligence Center

El proyecto explora cómo las noticias de código abierto y los conjuntos de datos contextuales pueden apoyar el monitoreo analítico de las narrativas de conflictos geopolíticos (Irán, Israel, EE. UU.).

El objetivo NO es la predicción de escaladas en el mundo real o la atribución causal, sino la exploración estructurada de señales de información pública a través de Machine Learning.

Fuentes Analizadas

  • Google News & GDELT
  • Datos Geoespaciales (NASA FIRMS, ACLED)
  • Datos de Movimiento (UNHCR, WorldPop)

Arquitectura de Modelos ML

NLP Clásico

Limpieza, tokenización, remoción de stopwords y vectorización TF-IDF como línea base interpretable.

NLP Moderno

Embeddings con sentence-transformers y visualización semántica con proyección UMAP.

BERTopic

Descubrimiento de temas latentes mediante embeddings de transformers y c-TF-IDF.

Gradient Boosting

Clasificador ML supervisado con etiquetas débiles heurísticas (Accuracy ~91%).

Random Forest

Algoritmo de ensamble robusto para clasificación (Accuracy ~85%).

FAISS Indexing

Búsqueda semántica de similitud ultra-rápida de vecindarios semánticos.
